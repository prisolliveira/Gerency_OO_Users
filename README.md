# 👥 Gerency OO Users
> Sistema web de gerenciamento de usuários — ASP.NET MVC

---

## Sobre o projeto

O **Gerency OO Users** é uma aplicação web desenvolvida para gerenciar os usuários internos de uma empresa. O sistema conta com dois níveis de acesso: administrador, responsável pela manipulação completa dos dados, e usuário comum (funcionário), com acesso restrito a consultas. Ambos os perfis possuem autenticação via login.

---

## Perfis de acesso

| Perfil | Descrição |
|---|---|
| 🔐 Administrador | Cadastro, edição e remoção de usuários. Acesso total ao sistema. |
| 👤 Funcionário | Consulta de informações limitadas. Acesso somente leitura. |

---

## Funcionalidades

- ✅ Autenticação com login para ambos os perfis
- ✅ CRUD de usuários (administrador)
- ✅ Consulta de dados com acesso restrito (funcionário)
- ✅ Integração com API externa
- ✅ Interface web responsiva
